全国计算机考试详解

一、考试概述​

全国计算机等级考试（NCRE）是由‌教育部教育考试院‌主办的计算机应用能力测评体系，始于1994年，面向社会提供计算机应用能力的标准化认证‌:。

  • ‌考试性质‌：全国统一命题、统一考试，采用上机考试形式‌:。
  • ‌报考条件‌：不限年龄、职业、学历，考生可自主选择报考级别‌:。
  • ‌考试频次‌：2025年设3月、6月、9月、12月四次考期，其中3月和9月开考全部科目，12月仅开考一、二级科目‌:。

二、考试等级与科目设置​

（一）等级划分​

等级定位主要科目考试时长
‌一级‌操作技能级计算机基础及MS Office应用（代码14）、网络安全素质教育（代码17）等90分钟‌:
‌二级‌程序设计/办公高级应用级C语言程序设计、Python语言程序设计、MS Office高级应用与设计（代码65）等120分钟‌:
‌三级‌工程师预备级数据库技术、网络技术等120分钟‌:
‌四级‌工程师级网络工程师、数据库工程师等90分钟‌:

（二）科目调整​

  • 2018年新增‌Python语言程序设计‌（二级代码66）‌:
  • 2019年起二级语言类科目要求‌选择题得分≥50%‌方可获证‌:

三、考试流程​

1. 报名阶段​

  • ‌报名入口‌：通过省级承办机构指定网站（如ncre-bm.neea.edu.cn）注册报名‌:
  • ‌材料要求‌：需上传有效身份证件（居民身份证/护照等）及电子照片‌:
  • ‌准考证打印‌：考前一周可下载打印‌:

2. 考试实施​

  • ‌考试形式‌：全部科目采用无纸化上机考试‌:
  • ‌成绩评定‌：等第制（优秀/良好/及格/不及格），二级科目需总分≥60且选择题达标‌:

3. 证书管理​

  • ‌证书发放‌：成绩合格者可申请电子证书，纸质证书由考点统一发放‌:
  • ‌补办流程‌：遗失证书可通过中国教育考试网申请补办合格证明‌:

四、考试特点与趋势​

  1. ‌实用导向‌

    • 一级MS Office考试要求掌握‌Office 2016版本‌操作‌:
    • 网络安全素质教育科目重点考察‌网络伦理与防护技能‌‌:

  2. ‌技术迭代‌

    • 2021年起二级MS Office考试系统升级至‌Office 2016‌‌:
    • 三级、四级科目强化‌工程实践能力‌考核‌:

五、注意事项​

  • ‌考试纪律‌：禁止携带通讯设备，违规者取消成绩‌:
  • ‌区域差异‌：6月、12月考试安排由省级承办机构自主决定，需关注当地公告‌:
  • ‌备考建议‌：建议通过‌官方模拟系统‌熟悉考试界面，避免版本差异影响操作‌:
